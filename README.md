# Humming-Bird

P5.jsで作成された音声ゲーム

## 環境
- 声の出せる環境
### 動作環境
- Microsoft Edge
- GoogleChrome
- firefox
- safari


## 遊び方

画面をクリックしてスタート  
録音画面にて「準備完了」が表示されたら録音する  
録音完了後クリックしてゲームスタート
鳥を自分の音声を利用して上下に動かし  
音階に合わせて流れてくる風船をキャッチしよう！  
3回風船を取り逃がしたらゲーム終了！
